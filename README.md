## üí° Why this exists

> [!TIP]
> Multi-language Data Structures and Algorithms repository with automated quality gates.

**Background:** The project serves as a centralized hub for optimized algorithmic solutions, providing standardized templates, complexity analysis, and automated validation for contributors participating in open-source challenges.

## ‚ú® Features

- **Automated PR Validation and Auto-merge**: Custom workflow validates code quality and complexity analysis before merging.
- **Support for 15+ Programming Languages**: Standardized implementations across compiled, interpreted, and functional paradigms.
- **Standardized Topic-based File Organization**: Consistent directory structure (Arrays, Sorting, Trees) across all languages.
- **Complexity Analysis and Documentation**: Every solution requires Big-O notation and structured documentation.
- **Hacktoberfest Automation**: Integrated labeling and validation for open-source contribution events.

## üöÄ Quick Start

> [!IMPORTANT]
> Ensure all PRs include a complexity analysis in the header comments to pass automated validation.

### Prerequisites

| Requirement | Version | Notes |
|-------------|---------|-------|
| GCC / G++ | 11+ | Required for C++ implementations |
| JDK | 8+ | Required for Java solutions |
| Python | 3.8+ | Used for scripts and algorithmic models |
| PHP | 7.4+ | Required for PSR-12 compliant logic tests |

> [!WARNING]
> Automated CI checks will fail if the file structure does not follow the `[Language]/[Topic]/[Solution]` hierarchy.

### Clone & Setup

```bash
git clone https://github.com/D4rk-Pho3nix/OpenSauce.git
cd OpenSauce
```

### Install & Build
```bash
# Example for C++ components
g++ -O3 solution.cpp -o solution

# Example for Haskell
ghc solution.hs
```

### Run
```bash
# Run compiled C++ binary
./solution

# Run Python scripts
python3 solution.py

# Run PHP test server
php -S localhost:8000
```

> [!NOTE] 
> For Debt Simplifier implementations, use the specific build flags provided in the C++ directory.

> [!TIP] 
> Contributions during Hacktoberfest are automatically tagged if they follow the contribution guidelines.

### Uninstall
```bash
rm -rf OpenSauce
```

## ü§ñ Usage

Users can explore specific language directories to find optimized solutions for common DSA problems. Each solution includes complexity analysis and example test cases.

**Running the Debt Simplifier in C++**
```bash
g++ DebtSimplifier.cpp -o debt_sim && ./debt_sim
```

**Running PHP Web-based Problem Solutions**
```bash
cd PHP/test_karunia && php -S localhost:8000
```
