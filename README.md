## ğŸš€ Quick Start

> [!IMPORTANT]
> Ensure all contributions include Time and Space complexity analysis in the header comments to pass CI validation.

### Prerequisites

| Requirement | Version | Notes |
|-------------|---------|-------|
| G++ Compiler | 9.0+ | Required for C++17 implementations |
| Python | 3.8+ | Used for core algorithm scripts |
| Node.js | Latest | For JavaScript/TypeScript solutions |
| PHP | 7.4+ | Required for the PHP test runner |

> [!WARNING]
> Manual modifications to automated PR labels may disrupt the auto-merge workflow.

### Clone & Setup

```bash
git clone https://github.com/D4rk-Pho3nix/OpenSauce.git
cd OpenSauce
```

### Build

```bash
g++ -std=c++17 [filename].cpp -o [output]
```

### Run

```bash
python3 [filename].py
node [filename].js
go run [filename].go
```

> [!NOTE]
> Every implementation is required to include Time and Space complexity in Big O notation within the comments.

> [!TIP]
> Use the PHP test runner to verify logic for web-based algorithmic modules locally.




## ğŸ¤– Usage

Users navigate to the directory of their chosen language, locate the algorithm by topic (e.g., /sorting), and execute the file using the appropriate language runtime or compiler. All implementations are designed to be standalone for educational purposes.

### Running a C++ algorithm with complexity analysis
```bash
g++ 169-Majority\ Element.cpp -o majority && ./majority
```

### Executing the PHP test suite
```bash
php -S localhost:8000
# Open http://localhost:8000/tests/run_all_tests.php in browser
```
